# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x.y   | :white_check_mark: |
| 1.x.y   | :x:                |
| 0.x.y   | :x:                |

## Reporting a Vulnerability

Reporting sensitive information may be done by emailing pr@pufferpanel.com.

Reports which do not contain a valid subject or desciption will be ignored.
